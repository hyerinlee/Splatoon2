# Splatoon 2
(본 레포지토리는 프로젝트용이 아닌 포트폴리오 참고자료용으로 소스코드만 포함되어 있습니다.)  
## 개요
스플래툰 2 모작 프로젝트입니다.  
- 플랫폼: PC(Windows)
- 개발인원: 1명
- 개발도구: Unity3D, Visual Studio, goormIDE, phpMyAdmin, AWS S3
